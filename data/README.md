# ПОДЗЕМЕЛЬЯ ПЁСИКИ - Анализ и обработка правил

## Описание проекта

Этот проект представляет собой детальный анализ правил настольной игры "ПОДЗЕМЕЛЬЯ ПЁСИКИ" с созданием структурированных данных для дальнейшей обработки.

## Созданные файлы

### 1. `podzemelja_chunks.jsonl`
**Полный JSONL со всеми чанками из файла, разбитыми по секциям**

- Формат: JSONL (JSON Lines)
- Количество записей: 85 чанков
- Структура: каждая строка содержит объект с полями `id`, `content`, `type`, `section`
- Типы контента: title, subtitle, metadata, narrative, section_title, instruction, rule, tip, objective, etc.
- Секции: main_title, game_info, introduction, components, setup, player_components, gameplay, explore, exit, sleep, room_effects, monsters, coins_artifacts, traps, wanderer, chest, potions, door, portal, round_end, game_end

### 2. `terminology_database.json`
**База терминов и спецификация игры**

Содержит:
- **Game Specification**: основная информация об игре (название, жанр, количество игроков, возраст, длительность)
- **Terminology**: словарь всех игровых терминов и механик
  - Game Components: описание всех компонентов игры
  - Game Mechanics: правила действий, боя, специальных элементов
  - Player Resources: ресурсы игроков (жетоны, зелья, квесты)
  - End Game: условия победы и подсчет очков
- **Named Entities**: персонажи, монстры, квесты с детальным описанием

### 3. `ner_database.json`
**База NER (Named Entity Recognition)**

Содержит структурированные данные для распознавания именованных сущностей:

- **Entities**: 
  - Game Title: название игры
  - Player Characters: 4 персонажа с их способностями
  - Monsters: 8 типов монстров с характеристиками
  - Quests: 4 типа квестов
  - Game Elements: все компоненты игры
  - Symbols: игровые символы (меч, лук, щит и т.д.)
  - Locations: игровые локации

- **Relationships**: связи между сущностями
  - Character Abilities: какие способности у каких персонажей
  - Monster Trophies: какие трофеи дают монстры
  - Quest Requirements: что требуется для выполнения квестов

- **Statistics**: статистика по сущностям и их распределению

## Основные характеристики игры

- **Название**: ПОДЗЕМЕЛЬЯ ПЁСИКИ
- **Жанр**: Фэнтезийная настольная игра
- **Количество игроков**: 2-4
- **Возраст**: от 8 лет
- **Длительность**: 30 минут
- **Раундов**: 3
- **Цель**: Набрать больше всего золотых монет

## Использование

Файлы созданы в форматах JSON и JSONL для легкой интеграции в различные системы обработки данных, машинное обучение и анализ текста. Структура данных позволяет:

1. Извлекать конкретные правила и механики игры
2. Анализировать взаимосвязи между игровыми элементами
3. Обучать модели распознавания игровых терминов
4. Создавать интерактивные руководства и помощники

## Структура данных

Все файлы используют стандартные форматы JSON/JSONL с четкой иерархией и именованием полей для обеспечения совместимости и расширяемости.
